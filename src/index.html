Welcome to DevServ

Just punch in the directory path of your project, and a development server will automatically spin up for you.
It's even got 'live reload', so whenever you make an edit to your project files, the browser will automatically refresh!

An easy to get the path directory on Macs is to open Finder, select your project folder, then press Command+Option+C. Now it's copied and you can paste it in.

To shut this server down, just quit this app.
